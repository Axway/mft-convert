### iblksize

#### COPYFILE

****[IBLKSIZE = {block size of the
input file &#124; n}]****

****{0...65535}****

Defines input file block size in bytes. The value must be greater than
the value of the [ilrecl](../ilrecl) parameter.

The table indicates  if the iblksize parameter
is used.


| OS  | IBLKSIZE  |
| --- | --- |
| MVS (z/OS) | YES  |
| OS400  | YES  |
| UNIX  | NO  |
| VMS  | NO  |
| Windows  | NO  |
