### lowport

#### CFTNET

{{% TransferCFT/snippets/obsolete%}}

****[LOWPORT = {5000 &#124; n}]****

TCP/IP only.

Defines the lower limit for the range of outgoing ports (value included).

- Possible
    values: 0 to 65535  
    Selecting 0 enables the system
    to select the outgoing port number
- Default
    value is 5000
