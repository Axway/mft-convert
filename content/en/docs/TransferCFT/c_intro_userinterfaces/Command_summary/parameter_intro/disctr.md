### disctr

#### CFTPROT

****[disctr = <u>45</u> &#124; n } ]    {0...3600}****

Network disconnection wait timeout (default = 45). Wait
timeout, in seconds, for the partner in the connection before disconnection,
after sending an ABORT request.
