### tcp

TCP
= { CFT
&#124; <u>OFTP</u>} ]

Processing method used for protocol messages:

- CFT: method specific to {{< TransferCFT/suitevariablesTransferCFTName >}}
- OFTP: standard method (RFC 2204)

This value applies in both initiator and responder mode.
