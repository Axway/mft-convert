#### WLOG

****logchg = { NONE &#124; STATIC &#124; DYNAMIC &#124; ALL
}****

Log configuration changes:

- none - no logging
    is performed
- static - only the
    static configuration is logged
- dynamic - only
    the dynamic configuration is logged
- all - log all changes
