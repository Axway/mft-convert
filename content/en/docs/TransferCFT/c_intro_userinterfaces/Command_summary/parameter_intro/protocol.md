### protocol

####  

#### CFTNET TYPE = TCP

<span style="font-weight: bold;">****PROTOCOL = { '  '
| SOCKS4| SOCKS5 }****</span>

Defines the remote TCP network resource proxy used to access the second
network:

- SOCKS4
- SOCKS5
- ' ' a blank indicates no proxy protocol is defined

 

#### CFTCOM TYPE=TCPIP

****PROTOCOL =string****
