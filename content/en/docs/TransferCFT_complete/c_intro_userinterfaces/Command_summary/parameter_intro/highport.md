### highport

#### CFTNET, TYPE = TCP

{{% TransferCFT/snippets/obsolete%}}

****[HIGHPORT = {65535 &#124; n}]****

Defines the upper limit for the range of outgoing ports (value included).

- Possible
    values: 1 to 65535
- Default
    value is 65535.
