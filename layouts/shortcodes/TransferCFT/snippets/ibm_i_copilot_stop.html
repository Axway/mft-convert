****Menu**** 1. Access the *Transfer CFT* **Main Menu**. In the Main Menu enter the command `cft` and press ****Enter**** to open the Transfer CFT menu. 1. Enter **1** to access **Common CFT commands**. 1. Select option ****2**** ****Stop Copilot****. Only the server waiting for a connection is stopped.
