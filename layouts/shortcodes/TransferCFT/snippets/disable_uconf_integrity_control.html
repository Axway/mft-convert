To disable integrity control for the UCONF dictionary, add the `-mac=no` parameter to the `cftruntime `command either when creating the runtime environment or regenerating the uconf file: Unix syntax: - `cftruntime [-profile|-n --uconf --mac=no` - `cftruntime [-profile|-n --inst --mac=no` Windows syntax: - `cftruntime [-profile|-n -uconf -mac=no` - `cftruntime [-profile|-n -inst -mac=no`
