****[SCOPE = { PARENT | CHILDREN | ALL } ]**** - PARENT: Applies the command to the selected transfers. - CHILDREN: Applies the command to the children of the selected transfers if the selected transfers are generic. Otherwise: - If the selected transfers are not generic, then the command is performed on the selected transfers themselves - If the selected transfers are generic parents, the command is NOT performed on these transfers - ALL: Applies to the selected transfers and all of their children.
